# Dotfiles

This is my personal dotfiles repo. You may use it or get inspired by it.

## Usage

Clone this repo and run the installer:

```bash
git clone https://github.com/rgruyters/dotfiles.git $HOME/dotfiles
cd $HOME/dotfiles
./install.sh
```
